# 📊 Netflix Streaming Data Analysis

## Overview
This project analyzes user streaming behavior using a Netflix-style dataset to identify viewing patterns, user engagement trends, and basic predictive insights.

---

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL (MySQL)
- Jupyter Notebook
- Scikit-learn (Linear Regression)

---

## Data Description
The dataset includes:
- User data: age, gender, country, subscription type
- Content data: title, genre, duration, release year
- Viewing data: watch time, device type, watch date

---

## Key Work Done

### Data Preprocessing
- Cleaned and structured raw data
- Merged multiple datasets (user, content, viewing)
- Created new features:
  - Age groups
  - Time-based features (month, year, weekday)

---

### Exploratory Data Analysis
- Top watched titles analysis
- Genre popularity analysis
- Watch time by device type
- Watch time by subscription type
- Watch time by age group
- Content duration distribution

---

### Trend Analysis
- Daily watch time trend
- Monthly watch time trend
- Identified user engagement patterns over time

---

### Predictive Analysis
- Built a Linear Regression model to forecast future watch time
- Used time-based features for trend prediction

---

## Key Insights
- User engagement changes over time, showing behavioral patterns
- Certain genres and devices contribute more to watch time
- Subscription type impacts viewing behavior
- Forecast indicates a slight stabilization or decline in watch time
